## How to Run the Code

You can run these programs in multiple ways:

### 1. Using a Browser (HTML + JS)

1. Open any `.html` file from the `Loops` folder.
2. Right-click and select **Open with ‚Üí Browser** (Chrome, Edge, etc.).
3. Open **Developer Tools ‚Üí Console** (`F12` or `Ctrl + Shift + I`) to see the output if it is logged using `console.log()`.

### 2. Using the Browser Console Only

1. Open your browser.
2. Press `F12` ‚Üí go to the **Console** tab.
3. Copy the JavaScript code from the `.js` file.
4. Paste it into the console and press **Enter**.

### 3. Using VS Code (Recommended)

1. Open this project folder in **VS Code**.
2. Install the **Live Server** extension (optional but helpful).
3. Right-click on an `.html` file ‚Üí **Open with Live Server** to run it in the browser.
